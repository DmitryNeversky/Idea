Description

Individual project which was used as Pet-project. It hasn't found an appointment, so It's decided to open that for a public. I found it useful for processing requests
from employee inside a company. Wich request has status(accepted, looking, declined). Before using the system each user must be registered.
Btw, there is a cool interface :D I spent on it much time and there is adaptation existing for every device as well.

Applied technologies

Frontend: Angular 12 - is used for frontend development.

Backend: Spring Boot - is used for backend development.

Broker: RabbitMQ - is used for communication between services.

Database: MySQL - is used for storing data.

Json Web Token - is used for providing authentication and authorization.

Installation using Docker Compose

- download docker-compose.yaml configuration file

- change environment variables for your own (optional)

- go to the folder with the downloaded file

- open a console and run the command docker-compose up -d (-d means detach mode)

- as soon as application is started you may go to a browser on http://localhost:4200/ page

- in order to shutdown the application run the command docker-compose down in the same folder
